## Technical Approach

### New Files

```
src/text2image/
├── client.py           (existing)
├── image_utils.py      (existing)
├── describer.py        (NEW — ImageDescriber protocol + two implementations)
└── __init__.py

cli.py                  (refactor to Click group)
```

### `describer.py` — Adapter Layer

```
ImageDescriber (ABC)
  ├── describe(image_path, prompt) -> DescribeResult

KimiCodeDescriber(ImageDescriber)
  - subprocess.run(["kimi", "--print", "-p", prompt_with_image, ...])
  - timeout from config
  - parse stdout as description text
  - handle exit codes 0/1/75

OpenRouterVisionDescriber(ImageDescriber)
  - reuses existing OpenAI client (OpenAI(base_url=openrouter))
  - calls chat.completions.create with vision model
  - builds image_url content block (reuse image_utils.encode_image_to_data_url)
  - returns description from response.choices[0].message.content
```

### `cli.py` — Refactor

Current flat `@click.command()` → `@click.group()` + two subcommands:

```
@click.group()
def cli(): ...

@cli.command("generate")
def generate(...): ...   # all current options, unchanged

@cli.command("describe")
@click.argument("image")
@click.argument("prompt", required=False)
@click.option("--backend", type=click.Choice(["kimi", "openrouter"]), default="kimi")
@click.option("--timeout", type=int)
@click.option("--json", "json_output", is_flag=True)
def describe(image, prompt, backend, timeout, json_output): ...
```

### Configuration Resolution

```
KIMI_PATH      → --kimi-path CLI > KIMI_PATH env > "kimi"
DESCRIBE_TIMEOUT → --timeout CLI > DESCRIBE_TIMEOUT env > 60
API_KEY        → --api-key CLI > OPENROUTER_API_KEY env > .env
```

### Key Implementation Notes

- `KimiCodeDescriber` builds a kimi print command: `kimi --print --final-message-only -p "<prompt> [image: <path>]"`
- `OpenRouterVisionDescriber` uses the same pattern as existing `TextToImageClient._build_messages()` for image encoding
- `ImageDescriber.create(backend)` factory selects implementation
- Both backends return `DescribeResult` with same structure
