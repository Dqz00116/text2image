# Design: .env Configuration for All CLI Defaults

## Approach

All Click option defaults change from literal to `None`. Resolution at function body:

```
CLI flag (if not None) > os.environ.get(VAR) > hardcoded fallback
```

`load_dotenv()` already called at function entry, so `.env` values are loaded into `os.environ` before resolution.

## No alternative needed

This is a find-and-replace style mechanical change following the existing pattern already used for `--api-key` and `--timeout`.
