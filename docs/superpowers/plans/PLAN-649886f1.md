## Test Strategy

| What | How | Where |
|------|-----|-------|
| Env var overrides default | `monkeypatch.setenv` + invoke without flag | `test_cli.py` (integration) |
| CLI flag overrides env var | set env var + pass CLI flag | `test_cli.py` |
| Fallback to hardcoded default | `monkeypatch.delenv` + invoke without flag | `test_cli.py` |
