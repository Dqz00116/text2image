## Tasks

### Phase 1: describer.py (TDD)

- [ ] TASK-001: Create `tests/unit/test_describer.py` with `DescribeResult` TypedDict test, then create `src/text2image/describer.py` with `DescribeResult` and `ImageDescriber` ABC
- [ ] TASK-002: Add `TestKimiCodeDescriber` — test `describe()` calls correct subprocess args, parses stdout, handles exit codes 0/1/75, handles timeout
- [ ] TASK-003: Implement `KimiCodeDescriber` — `subprocess.run` with `kimi --print --final-message-only -p`, capture stdout, map exit codes, `KIMI_PATH` env resolution
- [ ] TASK-004: Add `TestOpenRouterVisionDescriber` — test `describe()` builds correct messages with image_url, calls mocked OpenAI client, extracts content
- [ ] TASK-005: Implement `OpenRouterVisionDescriber` — reuses `encode_image_to_data_url`, calls `chat.completions.create` with vision model, extracts response text
- [ ] TASK-006: Add `test_create_describer` — factory returns correct backend class, raises on invalid backend
- [ ] TASK-007: Implement `create_describer(backend, **kwargs) -> ImageDescriber` factory

### Phase 2: CLI refactor + describe command

- [ ] TASK-008: Refactor `cli.py` — convert `@click.command()` to `@click.group()`, move current command to `generate` subcommand, verify `cli.py generate --help` works
- [ ] TASK-009: Update `tests/integration/test_cli.py` — all CLI invocations use `["generate"]` subcommand, all tests pass
- [ ] TASK-010: Add `describe` subcommand — `@cli.command("describe")` with `IMAGE` (arg), `PROMPT` (arg, optional), `--backend`, `--timeout`, `--json`
- [ ] TASK-011: Create `tests/integration/test_describe_cli.py` — mock `KimiCodeDescriber.describe`, test `cli.py describe img.png`, `cli.py describe img.png "question"`, `--json`, `--backend openrouter`
- [ ] TASK-012: Wire `describe` command — resolve timeout from `--timeout` / `DESCRIBE_TIMEOUT` env / 60, call describer, format output (text or JSON)

### Phase 3: Verify

- [ ] TASK-013: Run `uv run pytest` — all tests pass, no regressions
- [ ] TASK-014: Run `uv run python cli.py generate --help` and `uv run python cli.py describe --help` — manual smoke test

## Test Strategy

| What | How | File |
|------|-----|------|
| `DescribeResult` TypedDict shape | import verify | `test_describer.py` |
| `KimiCodeDescriber.describe()` | mock `subprocess.run` | `test_describer.py` |
| `KimiCodeDescriber` error paths | mock exit codes 1, 75, timeout | `test_describer.py` |
| `OpenRouterVisionDescriber.describe()` | mock `OpenAI.chat.completions.create` | `test_describer.py` |
| `create_describer` factory | smoke: returns correct class | `test_describer.py` |
| `generate` subcommand | existing integration tests | `test_cli.py` |
| `describe` subcommand | mock `ImageDescriber.describe`, check output | `test_describe_cli.py` |
